0.总有人给文件起名忘写.c
1.也总有人中英文标点不分
2.#include忘写"#"
3.不要缺少头文件
4.int main()尽量加上void
5.同理在结尾写上return 0
6.一行语句结尾记得加分号
7.C语言关键字不做标识符
8.用以分隔,符号和空格至少有一个
9.如果可以,尽量显式变量类型转换
10.变量初始化:
    定义但不赋值,可以逗号分隔;
    既定义又赋值,最好一个一个写;
    例如:int x,y,z=1只给z赋值
11.不论#define还是const都要大写
12.char:缺省为'\0'(空字符)而不是'0'(零)
13.0x表示十六进制,0表示八进制,都是数字'0'
14.浮点常量的指数形式'数字+E+数字',E前后不能有空格
15.字符常量用单引号,字符串常量用双引号
16.字符常量不能空,字符串常量可以空
17.多条语句要用{}黏结为代码块
18.for(初始化;条件;循环体),注意顺序与分隔符
19.break和continue只能跳出一层循环
20.for中的continue执行后自增语句仍然会执行
21.switch语句的case后面不要加括号
22.switch的每个case后面都要加break
23.非main函数内避免声明全局变量
24.数组的下标从0开始
25.数组的声明:
    int a[2];//正确
    int b[]={1,2};//正确
    int c[];//错误,未初始化
26.函数不能返回数组,但可以返回指向数组的指针
    例如下面的代码:
    #include<stdio.h>
    int* func(int a,int b)
        {static int c[2];
        c[0]=a;
        c[1]=b;
        return c;}
    int main(void)
        {int *p=func(1,2);
        printf("%d %d",p[0],p[1]);
        return 0;}
